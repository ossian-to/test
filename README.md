# git 簡單教學

## 先 Clone 遠端倉到本地

- 先到指令目錄
- 下指令 git clone https://github.com/ossian-to/test.git

## 指定本地倉的遠端倉路徑

  - git remote add origin 遠端倉的路徑

  如果使用 clone 的話，就不需要指令遠端倉的路徑，也不需要對本地倉的目錄初始化
## 常用指令

- 第零步：查看目前 git 的狀態 git status
- 第一步：初始化 git init
- 第二步：建立索引 git add .
- 第三步：commit 到本地倉 git commit -m "文字註解"
- 第四步：推送到遠端倉 git push
